# 🚂 Mexican Train Tracker

A mobile-first web app for tracking Mexican Train Domino games.

**Live app:** https://YOUR_USERNAME.github.io/mexican-train/

## Features

- 2–8 players with real game train colors
- Domino engine display per round
- Round-by-round score entry
- Race graph (ranking & cumulative) with landscape support
- Full rules reference with contextual tips
- Auto-saves to phone memory — survives lock screen & refresh
- Edit any past round score
- Reset button to start fresh

## Stack

- React + Vite
- Barlow Condensed + Inter fonts
- localStorage for persistence
- GitHub Actions → GitHub Pages

## Deploy your own

1. Fork this repo
2. Go to **Settings → Pages → Source → GitHub Actions**
3. Push to `main` — it deploys automatically

## Local dev

```bash
npm install
npm run dev
```
